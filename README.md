# 🕹️ Retro Tech Store

A simple Node.js-powered retro-themed tech store that serves a responsive frontend using Node's built-in HTTP module. The project focuses on learning the fundamentals of backend development, including creating an HTTP server, serving static files, and understanding how Node.js works without using frameworks like Express.

---

## 🚀 Features

- 🖥️ Retro-inspired user interface
- 📦 Node.js HTTP server
- 📄 Serves HTML, CSS, and JavaScript files
- ⚡ Lightweight and beginner-friendly
- 🎨 Responsive design
- 🔧 Built without Express to understand core Node.js concepts

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Node.js
